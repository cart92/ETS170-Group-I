The application that is to be developed is supposed to increase the shopping malls customer satisfaction, the number of visiting customer and increase the number of sales the stores in the mall has.\\\\

\textit{"To earn cash" - David Cartbo.}

%==================================

\section{Terminology}
	\begin{description}
	\item[The app] is referring to the ordered application gör a shopping mall, i.e. the ordered product.
	\item[The user] is referring to the person using the application on their device.
	\item[Device] is referring to the unit the user is running the app on, e.g. mobile phone and smart-watch.
\end{description}
